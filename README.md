# MEV Guard

AI-powered MEV attack detector and loss tracker for Ethereum wallets.

## Demo

This is a static demo version with mock data. Try scanning `vitalik.eth` or any wallet address.

## Features

- Wallet scanning for MEV attacks
- Sandwich attack detection
- AI-powered analysis (Claude)
- Loss calculation
- Protection recommendations

## Run Locally

```bash
npm install
npm run dev
```

## Full Version

The complete version with real blockchain scanning is in the `main` branch. Requires:
- Alchemy API key
- Anthropic API key

## Tech Stack

- React + TypeScript
- TailwindCSS
- Vite
